# launch:

quickly install:
- python & packages [bottle,jinja2,honcho,gunicorn]
- nodejs & packages [forever,http-proxy,passport,express,consolidate,swig]

**Prerequisites:**
- Mac:
 - Xcode
 - OS X 10.7-10.8

**Supported Systems:**
- Ubuntu
- CentOS
- Opensuse
- Fedora
- Mac

to run:
`bash <(curl -fsSL https://raw.github.com/gabeio/launch/install)`

**Tested on:**
- ubuntu (12.04)

**Soon to be tested on:**
- OS X 10.8
- Fedora
- Opensuse
- CentOS
- (and more)
