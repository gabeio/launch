# launch:

quickly install:  
python & packages [bottle,jinja2,honcho,gunicorn]  
nodejs & packages [forever,http-proxy,passport]  

**Supported Systems:**
- Ubuntu
- CentOS
- Opensuse
- Fedora
- Mac

to run:
`bash <(curl -fsSL https://raw.github.com/gabeio/launch/install)`

**Tested on:**
- ubuntu (12.04)
